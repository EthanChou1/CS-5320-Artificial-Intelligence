Introduction¶

In this assignment you will implement a simulator environment for an automatic vacuum cleaner robot, a set of different reflex-based agent programs, and perform a comparison study for cleaning a single room. Focus on the cleaning phase which starts when the robot is activated and ends when the last dirty square in the room has been cleaned. Someone else will take care of the agent program needed to navigate back to the charging station after the room is clean.

PEAS description of the cleaning phase¶

Performance Measure: Each action costs 1 energy unit. The performance is measured as the sum of the energy units used to clean the whole room.

Environment: A room with $n \times n$ squares where $n = 5$. Dirt is randomly placed on each square with probability $p = 0.2$. For simplicity, you can assume that the agent knows the size and the layout of the room (i.e., it knows $n$). To start, the agent is placed on a random square.

Actuators: The agent can clean the current square (action suck) or move to an adjacent square by going north, east, south, or west.

Sensors: Four bumper sensors, one for north, east, south, and west; a dirt sensor reporting dirt in the current square.

The agent program for a simple randomized agent¶

The agent program is a function that gets sensor information (the current percepts) as the arguments. The arguments are:

  • A dictionary with boolean entries for the for bumper sensors north, east, west, south. E.g., if the agent is on the north-west corner, bumpers will be {"north" : True, "east" : False, "south" : False, "west" : True}.
  • The dirt sensor produces a boolean.

The agent returns the chosen action as a string.

# The simulation environment
def sim_environment(agent, max_steps, gridsize, verbose=True):
    # Initialize the 5x5 grid with dirt (1 for dirty, 0 for clean) using a 20% probability
    grid_size = gridsize
    grid = np.random.choice([0, 1], size=(grid_size, grid_size), p=[0.8, 0.2])

    # Starts agent at random
    pos = [np.random.randint(0, grid_size), np.random.randint(0, grid_size)]

    # Performance measure: count the number of actions taken
    action_count = 0

    for step in range(max_steps):
        # Get the bumper sensor inputs (True if agent is at the boundary)
        bumpers = {
            "north": pos[0] == 0,
            "south": pos[0] == grid_size - 1,
            "west": pos[1] == 0,
            "east": pos[1] == grid_size - 1
        }

        # Check if the current square is dirty
        dirty = grid[pos[0], pos[1]] == 1

        # Call the agent function to get the next action
        action = agent(bumpers, dirty)
        if verbose:
            print(f"Step {step}: Agent at {pos}, action: {action}")

        if action == "suck":
            # If dirty, clean square
            if dirty:
                grid[pos[0], pos[1]] = 0
                if verbose:
                    print(f"Square cleaned at {pos}")
        elif action == "north" and not bumpers["north"]:
            pos[0] = pos[0] - 1
        elif action == "south" and not bumpers["south"]:
            pos[0] = pos[0] + 1
        elif action == "west" and not bumpers["west"]:
            pos[1] = pos[1] - 1
        elif action == "east" and not bumpers["east"]:
            pos[1] = pos[1] + 1

        # Increase action count by 1
        action_count += 1

        # Check if the grid is fully cleaned
        if np.sum(grid) == 0:
            if verbose:
                print(f"Room cleaned in {action_count} actions.")
            break

    return action_count

#Description of the states

States:

  • Current Location: The agent keeps track of its position in the room
  • Visited square: The agent remembers which squares it already visited
  • Wall locations: The agent's bumpers help it figure out where the walls are

The agent moves to a corner to determine its location and then it can navigate through the whole room and clean dirty squares in a zigzag pattern.

The agent starts in a random location. It will head to the Northwest corner, cleaning any dirty square along its way. Once it reaches the northwest corner, it will begin the zigzag pattern. If the current square it is on is dirty, it will clean. If the square is clean, the agent will either move east or west. Once it reachs either the east or west wall, it will head south one square before heading in the opposite direction, so if it hit the east wall, it will go down one square then start heading west.

# Your code goes here
class Agent:
    def __init__(self, initial_position = (0,0), name = "Agent"):
        self.position = list(initial_position) # this is the state of the agent
        self.name = name
        self.direction = "west" # make sure it heads to NW corner
        self.found_corner = False # checks if corner has been reached

    def act(self, bumpers, dirty):
        if dirty:
            return "suck"

        if not self.found_corner:
            if bumpers["west"]:
                if bumpers["north"]:
                  self.found_corner = True # Found NW corner
                  self.direction = "east" # starts moving
                else:
                  self.position[0] = self.position[0] - 1
                  return "north"
            else:
                self.position[1] = self.position[1] - 1
                return "west"

        # Begins heading east uuntil hits wall, then goes south one square
        if self.direction == "east":
            if bumpers["east"]:
                self.direction = "west"
                if not bumpers["south"]:
                  self.position[0] = self.position[0] + 1
                  return "south"
            else:
              self.position[1] = self.position[1] + 1
              return "east"

        #Begins heading west until hits wall, then goes south one square
        if self.direction == "west":
            if bumpers["west"]:
                self.direction = "east"
                if not bumpers["south"]:
                  self.position[0] = self.position[0] + 1
                  return "south"
            else:
              self.position[1] = self.position[1] - 1
              return "west"

        return "stay"
